# syslab

Учебная консольная программа, демонстрирующая работу с системными вызовами POSIX.

## Сборка

```sh
make
```

Будет собран бинарный файл `./syslab`.

## Использование

```
syslab <команда> [аргументы]
```

### `info <путь>`
Выводит информацию о файле. Используемые системные вызовы: `stat`, `access`.

```
$ ./syslab info file.txt
Path: file.txt
Type: regular file
Size: 1248 bytes
Readable: yes
Writable: yes
Executable: no
```

### `copy <исходный> <целевой>`
Копирует содержимое файла. Используемые системные вызовы: `open`, `read`, `write`, `close`.

```
$ ./syslab copy a.txt b.txt
```

### `tail <файл> <N>`
Выводит последние `N` строк файла. Чтение ведётся блоками с конца файла. Используемые системные вызовы: `open`, `lseek`, `read`, `write`, `close`.

```
$ ./syslab tail /var/log/syslog 20
```

### `run <программа> [аргументы...]`
Создаёт дочерний процесс и запускает в нём указанную программу. Используемые системные вызовы: `fork`, `execvp`, `waitpid`.

```
$ ./syslab run ls -la /tmp
```

Код возврата `syslab` соответствует коду возврата дочернего процесса.
