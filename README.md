# syslab

Учебная консольная программа, демонстрирующая использование системных вызовов POSIX.

## Сборка

```sh
make
```

Получится исполняемый файл `./syslab`.

## Подкоманды

### `syslab info <путь_к_файлу>`

Выводит информацию о файле, используя `stat(2)` и `access(2)`.

```
$ ./syslab info syslab.c
Path: syslab.c
Type: regular file
Size: 1248 bytes
Readable: yes
Writable: yes
Executable: no
```

### `syslab copy <исходный_файл> <целевой_файл>`

Копирует содержимое одного файла в другой через `open(2)`, `read(2)`, `write(2)`, `close(2)`.

```
$ ./syslab copy a.txt b.txt
```

### `syslab tail <файл> <N>`

Выводит последние `N` строк файла. Реализовано через `open`, `lseek`, `read`,
`write`, `close` — файл сканируется блоками с конца, пока не наберётся `N`
переводов строки.

```
$ ./syslab tail /etc/hosts 3
```

### `syslab run <программа> [аргументы...]`

Запускает указанную программу в дочернем процессе через `fork(2)` +
`execvp(3)` и ожидает её завершения с помощью `waitpid(2)`. Код возврата
программы становится кодом возврата `syslab`.

```
$ ./syslab run ls -la /tmp
```
